# GIPC 홈페이지 업로드 안내

## 포함 파일
- index.html : 메인 홈
- jesus-walk.html : 예수 동행 훈련원
- gips.html : GIPS & 통합 양육 포털
- dashboard.html : 세계 기도 대시보드
- mission.html : 선교 동원 허브
- oneness.html : 중보 기도 공동체
- resources.html : 리소스 & 행정
- mypage.html : 나의 기도방
- assets/styles.css : 전체 디자인
- assets/site.js : 메뉴/버튼 동작

## 업로드 방법
1. 압축을 풀어 모든 파일을 호스팅의 public_html 또는 www 폴더에 업로드하세요.
2. 기존 index.html이 있다면 백업 후 교체하세요.
3. 브라우저에서 https://gipmgroup.net/ 접속 후 확인하세요.

## 개인 운영 방법
- 글 수정: 각 .html 파일에서 한글 문구를 수정
- 색상 수정: assets/styles.css의 :root 색상값 수정
- 메뉴 수정: 각 파일 상단 header 영역 수정
- 새 자료 업로드: resources.html에 링크 추가
